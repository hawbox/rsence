
Himle RIA Server

Requirements:
 - Depends on ruby version 1.8 or later
 - Depends on a recent trunk build of Himle RIA Client
 - Confirmed to run on Mac OS X 10.4
 - Confirmed to run on Debian Linux / sid

Installation:
 1) Edit conf/server_config.rb to match your environment
 2) Run the server by starting bin/server.rb
