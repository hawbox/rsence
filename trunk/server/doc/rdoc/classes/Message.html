serve_rsrc( rsrc_data, content_type='text/plain' )

Sends any binary to be served, returns a static uri.

IMPORTANT: PLEASE call unserve_rsrc manually, when you don‘t need the resource anymore! Otherwise TicketServe will keep on chugging more memory every time you serve something.

HINT: Usually, it‘s a better idea to use serve_img or serve_file instead.
